# How to run
1. Download Python 3.9.
2. Clone the repo
3. CD into the directory
4. Run the following command: 
`pip install -r requirements.txt`
5. Run: `python app.py`
6. Visit [http://localhost:6969](http://localhost:6969)

# Tech stack:
1. Flask
2. PostgreSQL
3. TailwindCSS
4. JavaScript
5. AJAX
6. Vercel Edge Functions
